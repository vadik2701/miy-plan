МІЙ ПЛАН

Це мобільний планувальник для доставок, робочих справ, зустрічей,
дзвінків і особистих нагадувань.

Як додати запис:
1. Натисніть "Додати".
2. Вставте все повідомлення з Viber або напишіть завдання вручну.
3. Виберіть тип, дату та час.
4. Увімкніть "Нагадати сповіщенням" і дозвольте сповіщення браузеру.
5. Виберіть, як часто повторювати нагадування. Воно повторюватиметься,
   доки запис не буде позначено виконаним.

Коротку назву можна не вводити: вона автоматично береться з першого
рядка вставленого тексту.

Швидкий запуск на комп'ютері:
1. Відкрийте PowerShell у цій папці.
2. Виконайте: python -m http.server 8080
3. Відкрийте у браузері: http://localhost:8080

Щоб користуватися на телефоні:
- папку потрібно опублікувати на будь-якому HTTPS-хостингу;
- відкрийте адресу на телефоні;
- у меню браузера виберіть "Додати на головний екран".

Важливо:
- записи зберігаються локально у браузері конкретного пристрою;
- після входу через email і пароль записи синхронізуються через Firebase;
- на ПК та iPhone потрібно використати один і той самий акаунт;
- регулярно використовуйте "Завантажити резервну копію";
- якщо очистити дані браузера без резервної копії, записи буде втрачено.
- браузер може показувати нагадування, поки застосунок відкритий або
  працює у фоні; після примусового закриття браузер телефона може
  призупинити нагадування до наступного відкриття застосунку.

Налаштування Firebase:
1. Authentication -> Sign-in method -> Email/Password -> Enable.
2. Authentication -> Settings -> Authorized domains:
   додати vadik2701.github.io.
3. Firestore Database -> Create database -> Production mode.
4. У вкладці Rules вставити вміст файлу firestore.rules та Publish.
